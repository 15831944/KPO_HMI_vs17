The project file LF_Local is used for this local LF solution. It takes the SMM libraries from the Drive V:
after a local build of the libraries. The LF solution uses this project to give the possibility to debug 
the libraries in case of problems.

The project LF is used for the solution XPACT_SMC_BUILD and takes the libraries from $(X-Pact)/packages/models/SMM/lib
as used for normal distribution of well proofed and stable model libraries.
